# Code of Conduct

## Our Pledge

We want this project to be a respectful, inclusive space for everyone.

## Expected Behavior

- Be constructive and specific in feedback.
- Assume good intent and ask clarifying questions.
- Respect different experience levels and communication styles.

## Unacceptable Behavior

- Personal attacks, harassment, or discrimination.
- Trolling, hostile language, or repeated dismissive behavior.
- Publishing private information without permission.

## Scope

This applies to repository discussions, issues, pull requests, and related project communication.

## Enforcement

If you observe a violation, open a private report to the maintainer.
Maintainers may remove content or restrict participation for behavior that harms the community.
